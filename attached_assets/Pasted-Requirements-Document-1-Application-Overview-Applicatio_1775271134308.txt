5. Exceptions & Edge Cases
Scenario	Handling
Login with incorrect credentials	Display error: Invalid email or password
Registration with existing email	Display error: This email is already registered
Password and confirm password mismatch	Display inline validation error
Unauthenticated access to Dashboard	Redirect to Login page
Contact form submitted with empty fields	Display required field validation errors
Resume file download fails (backend error)	Download button shows fallback message: Resume temporarily unavailable
Background particle/gradient effect fails to render	Fallback to a static dark gradient hero background
GitHub or coding profile link not in resume	Hide the corresponding link/card rather than showing a broken link
Project has no live demo link	Hide the Live Demo button for that project card
Profile image not provided	Display initials avatar as placeholder
